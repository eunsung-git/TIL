

# 마크다운 활용법

## 1. 제목(heading)

제목은 '#' 을 통해 구성, # 갯수로 제목의 레벨 표현 (1~6)

### 제목테스트(h3)

#### 제목테스트(h4)

##### 제목테스트(h5)

###### 제목테스트(h6)

## 2. 목록

목록은 순서 0 또는 순서 x

1.  순서 0
2. 순서 0 -2

* 순서가 없는 목록
* bullet point
  * tab 으로 레벨 다운
  * 작성하고
* shift+tab 으로 나가기

엔터로 나가기 성공!

## 3. 인용문

부등호 '>' 로 인용문 작성

## 5. 이미지

![마크다운 이미지](C:\Users\student\Desktop\마크다운 이미지.jpg)

* typora 활용시 '파일>환경설정>이미지' 에서 아래 설정 체크
  * 가능하다면 상대적 위치 사용
  * 온라인 이미지에도 의 규칙 적용
  * 로컬 이미지에 이 규칙을 적용
* 특별한 동작 없음을 원하는 폴더 구조로 작성
  * ex) `./image`
* 위와 같이 설정 시 온라인에서도 이미지가 깨지지 않음
