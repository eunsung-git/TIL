### HTML tag 정리

```html
##### text
<strong> </strong>  굵게 강조
<u> </u>             밑줄
<h1> </h1> ~ <h6> </h6>  제목

##### sentence & paragraph
<br>                 갯수만큼 줄바꿈(enter), No close
<p> </p>             단락

##### image
<img src='주소' width="100%">  이미지 삽입

##### 목차 추가
<li> </li>           목차(목록)
<ul> </ul>           목록 간 경계 나눔 (Unordered list)
<ol> </ol>           목록 간 경계 나눔 (Ordered list)

##### web 구조
<!doctype html>         html 문서 표시
<html> </html>          html 태그
<head> </head>          본문 설명
<title> </title>        webpage 제목
<meta charset="utf-8">  webpage 방식
<body> </body>          본문

##### link
<a href="url주소" target="_blank" title="url내용입력">링크생성text</a>


##### semantic tag
<header>   </header>
<nav>  </nav>
<aside>  </aside>
<section>  </section>
<article>  </article>
<footer>  </footer>
```
